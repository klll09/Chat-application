# Chatty Application

A real-time chat application built with a modern MERN stack (MongoDB, Express, React, Node.js). 

## Live Demo
Check out the live application here: https://chat-application-9yyx.onrender.com

## Features
* **Real-time Messaging:** Experience instant communication.
* **Secure Authentication:** User signup and login functionality.
* **State Management:** Efficient data handling using modern frontend tools.
* **Responsive Design:** Fully functional on desktop and mobile devices.

## Tech Stack
* **Frontend:** React, Vite
* **Backend:** Node.js, Express
* **Database:** MongoDB (via MongoDB Atlas)
* **Deployment:** Render

## Project Structure
* `/backend` - Node.js/Express server logic and API routes.
* `/frontend` - React application source code.

## Getting Started
To run this project locally:

1.  Clone the repository:
    `git clone https://github.com/klll09/Chat-application.git`
2.  Install dependencies for both backend and frontend:
    ```bash
    cd backend && npm install
    cd ../frontend && npm install
    ```
3.  Set up your environment variables (`MONGODB_URI`, `JWT_SECRET`, etc.) in the backend folder.
4.  Start the application.
